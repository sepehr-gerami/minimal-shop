Minimal Shop

یک فروشگاه آنلاین مینیمال ساخته‌شده با Next.js (App Router)، که از Server-Side Rendering و React Server Components برای نمایش سریع‌تر محصولات استفاده می‌کنه.

ویژگی‌ها


🛍️ نمایش لیست محصولات با دیتای واقعی از Fake Store API
⚡ رندر سمت سرور (SSR) با Streaming و Suspense برای بارگذاری سریع‌تر
🎨 طراحی ریسپانسیو با Tailwind CSS
✨ انیمیشن‌های روان با Framer Motion
❤️ امکان افزودن محصول به Wishlist
🛒 سبد خرید (Cart)
⭐ نمایش امتیاز و تعداد نظرات هر محصول
🔝 دکمه‌ی بازگشت به بالای صفحه (Scroll to Top)
📱 ناوبری واکنش‌گرا (Navbar) با منوی موبایل


تکنولوژی‌های استفاده‌شده

کاربرد          ابزار

Next.js                 فریم‌ورک اصلی (App Router+Turbopack)

React                   کتابخانه‌ی رابط کاربری

TypeScript              TypeScriptتایپ‌سیفتی  

Tailwind CSS            استایل‌دهی

Framer Motion           انیمیشن

Bootstrap Icons         آیکون‌ها   


منبع داده
داده‌ی محصولات از گرفته میشه  و این API عمومی به صورت  SSR  رندر  میشهfakestoreapi.com
